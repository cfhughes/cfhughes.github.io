SUMMARY 

I am a Systems Analyst and Developer familiar with Linux, Windows, Android, iOS, PHP, MySQL, Java, HTML and JavaScript. I also have some experience with Oracle, Python and C++. I specialize in building tools that make existing data more accessible and useful. I was interviewed by KOAT news in Albuquerque about a free app that I made that presents real time data about Albuquerque buses.

PROJECTS

ALBUQUERQUE WHERE’S THE BUS

I built Android and iOS apps that show real-time GPS data from the Albuquerque bus system. The data is downloaded, then stored on a virtual server using Memcache, then processed and served to the apps with PHP. Keeping the Virtual Server running smoothly has taught me a lot about Linux system administration.

NEW MEXICO RAIL RUNNER EXPRESS SCHEDULE

I built Android and iOS apps that show the schedule for the Rail Runner commuter train. The data is stored locally and users can select the stops they are traveling between to be shown only the relevant information.
